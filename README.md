 # MLOps Unit 1 Project

## Setup Instructions

1. Clone repository
git clone <repo-link>
cd mlops-unit1

2. Create virtual environment
python -m venv venv

3. Activate environment
Windows:
venv\Scripts\activate

Linux/Mac:
source venv/bin/activate

4. Install dependencies
pip install -r requirements.txt

5. Run project
python src/train_model.py
